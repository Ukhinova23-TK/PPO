Что такое SQL -
простыми словами, это язык программирования структурированных запросов (SQL, Structured Query Language), 
который используется в качестве эффективного способа сохранения данных, 
поиска их частей, обновления, извлечения из базы и удаления.




Примеры SQL :


1) Простая выборка
	1.1) Select ПОЛЕ(ПОЛЯ) from ТАБЛИЦА(Ы)
		1.1.1) Select Name from Sight (Вырать все имена из таблицы Sight)
		1.1.2) Select * from Sight	(Вырать все поля из таблицы Sight)
2) Выборка с условием 
	2.1) Select * from Sight where id<5 (Выберит все поля где значение поля ID больше 5)
3) Выборка с соединением результатов (Теория https://blog.rc21net.ru/sql-join-union/ Дальше пригодится у Павловой
	3.1) Select * from Houses h(Сокращения) join Sight s on s.ID=h.SightID 
	
4) Удаление (Теория http://www.sql-tutorial.ru/ru/book_delete_statement.html)
	4.1) Delte From ТАБЛИЦА where УСЛОВИЕ
	
5) Обновление данных (Теория https://sql-language.ru/update.html)
	5.1) Update таблица set поле=значение where условие
	5.2) update sight set name='3 дурака' where Position='Центр'
